# Digital Clock Python

## Project Overview

A simple digital clock application built using Python's datetime module.

The program displays the current time in a user-friendly format.

## Features

* Displays current time
* Uses datetime.now()
* Uses strftime()
* Real-time clock display

## Technologies Used

* Python
* Datetime Module

## Sample Output

Current Time:
14:35:21

## Learning Outcomes

* Working with time
* Formatting output
* Understanding datetime
